# CONTROL DE GASTOS

Proyecto listo para subir a Vercel.

## Qué incluye
- App React + Vite
- Guardado local en el navegador del iPhone
- Exportar e importar copia de seguridad en JSON
- Gráficos de ahorro e histórico por categorías
- Icono del cerdito
- Manifest básico para usarla como app en iPhone

## Cómo subirlo a Vercel
1. Descarga este proyecto y descomprímelo.
2. En Vercel, crea un proyecto nuevo.
3. Sube esta carpeta completa.
4. Vercel detectará automáticamente `package.json`.
5. Pulsa deploy.

## Cómo usarla en iPhone
1. Abre la URL en Safari.
2. Pulsa compartir.
3. Elige "Añadir a pantalla de inicio".
4. Se instalará como acceso directo con icono propio.

## Desarrollo local
```bash
npm install
npm run dev
```

## Build local
```bash
npm install
npm run build
```
